## 🧑‍💻 Sobre mí

Soy desarrollador Full Stack con fuerte enfoque en **ingeniería backend**. Me especializo en diseñar y construir **APIs RESTful** seguras, mantenibles y listas para producción.

- 🔒 Autenticación y autorización con **JWT** y middlewares personalizados
- 🗄️ Diseño de bases de datos con **PostgreSQL** (relacional) y **MongoDB** (no relacional)
- ⚙️ Optimización backend para **rendimiento, seguridad y escalabilidad**
- 🔁 Control de versiones con **Git/GitHub** y flujos de trabajo ágiles con **Scrum**
- 🧩 Implementación de ORM/ODM para capas de acceso a datos limpias
