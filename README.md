👨‍💻 Sobre mí
Soy desarrollador Full Stack especializado en Backend, enfocado en la construcción de APIs robustas, seguras y escalables.
Trabajo principalmente con Node.js y NestJS, desarrollando arquitecturas backend modulares con TypeScript, TypeORM y PostgreSQL, implementando autenticación mediante JWT y control de acceso mediante middlewares personalizados.
También tengo experiencia integrando frontends desarrollados con React y Next.js, consumiendo APIs REST y conectando aplicaciones completas end-to-end.
He participado en el desarrollo de proyectos como una API de e-commerce y aplicaciones full stack, trabajando con Git, metodologías ágiles y desarrollo colaborativo.
Estoy profundamente interesado en Inteligencia Artificial y automatización, y ya he comenzado a experimentar con proyectos personales que integran APIs y soluciones innovadoras.
