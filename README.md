### 🏋️ PowerGym — Sistema de Gestión de Gimnasio
> API backend + frontend completo para gestión de gimnasios con autenticación, membresías y reserva de clases.

**Stack:** `NestJS` `TypeScript` `PostgreSQL` `TypeORM` `JWT` `Passport.js` `Swagger` `React` `GSAP` `CSS Modules`

**Características principales:**
- 🔐 Autenticación JWT con guards y RBAC (roles: Admin, Coach, User)
- 📚 Documentación interactiva con Swagger
- 🏗️ Arquitectura modular NestJS (Users, Auth, Classes, Coach, Schedules)
- 🗃️ Repository Pattern para separación de responsabilidades
- 🌐 Frontend en React con animaciones GSAP + ScrollTrigger
- ☁️ Desplegado en Render + Supabase (PostgreSQL)
